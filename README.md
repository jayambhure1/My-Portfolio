# My Portfolio

A clean, responsive personal portfolio website built with HTML, CSS, and Bootstrap. This project showcases a developer portfolio layout with sections for hero, about, skills, experience, certifications, contact, and a fixed-top navigation bar.

## Features

- Responsive layout using Bootstrap 5
- Hero section with profile image and call-to-action buttons
- About section with background and summary content
- Skills section with animated progress bars
- Experience timeline and certifications
- Contact section with social links
- Custom styling from `assets/css/main.css`
- Lightweight Bootstrap asset setup using only `bootstrap.min.css` and `bootstrap.bundle.min.js`

## Project Structure

- `index.html` — main portfolio page
- `assets/css/main.css` — custom styles
- `assets/bootstrap/css/bootstrap.min.css` — Bootstrap CSS
- `assets/bootstrap/js/bootstrap.bundle.min.js` — Bootstrap JS bundle
- `assets/js/main.js` — custom JavaScript for interactions
- `assets/images/` — profile image assets

## How to Run

1. Open `index.html` in your browser.
2. Or use a local server for best results:
   - `Live Server` extension in VS Code
   - `python -m http.server 8000` and open `http://localhost:8000`

## Customization

- Update the content sections in `index.html`.
- Change colors, typography, and layout in `assets/css/main.css`.
- Replace the profile image in `assets/images/myprofile.jpeg`.

## Notes

This project is a static website and does not require a build step. It is ideal for showcasing a developer portfolio or résumé-style personal site.
